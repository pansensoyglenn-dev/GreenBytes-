1. List Comprehensions

List comprehensions provide a concise way to create lists. They are often faster and more readable than traditional for loops.

# Instead of this:
squares = []
for x in range(10):
    squares.append(x**2)

# Do this:
squares = [x**2 for x in range(10)]

2. Formatted String Literals (f-strings)

Introduced in Python 3.6, f-strings are the most efficient and readable way to format strings.

Pro Tip: You can even perform math or call methods directly inside the curly braces! 
name = "Gemini"
print(f"Hello, {name}!") # Clean and fast

4. Looping with Zip

Use the zip() function to iterate over multiple lists in parallel.

names = ["Alice", "Bob"]
scores = [85, 92]

for name, score in zip(names, scores):
    print(f"{name} scored {score}")
