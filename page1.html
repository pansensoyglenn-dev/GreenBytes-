Introduction to Python Strings

Strings in Python are one of the most fundamental and commonly used data types. A string is a sequence of characters enclosed in single quotes ('), double quotes ("), or triple quotes (''' or """). Strings are immutable, meaning once created, their content cannot be changed (though you can create new strings based on operations).

Key features:

  • Strings support indexing (accessing individual characters) and slicing.
  • They come with a rich set of built-in methods for manipulation.
  • Python supports Unicode, so strings can handle characters from any language.
  • Common operations include concatenation, repetition, formatting, searching, and more.

Strings are used everywhere: storing text, processing user input, file handling, web data, and more.

Below are 30 sample programs demonstrating various aspects of Python strings. Each includes a short description, the code, and the expected output.

s = "Hello, World!"
print(s)
Output:
Hello, World!
Explanation: This program demonstrates how to assign a sequence of characters to a variable using double quotes and display it using the print() function.

s1 = "Hello"
s2 = "World"
print(s1 + ", " + s2 + "!")
Output:
Hello, World!
Explanation: Concatenation uses the + operator to merge multiple strings into one continuous sequence.

s = "Python"
print(len(s))
Output:
6
Explanation: The len() function returns the total count of characters within a string, including spaces and symbols.

s = "Hello, World!"
print(s[0])      # First character
print(s[-1])     # Last character
print(s[7:12])   # Substring
Output:
H
!
World
Explanation: Indexing allows access to specific characters via position (starting at 0), while slicing ([start:end]) extracts a range of characters.

s = "python is fun"
print(s.upper())
print(s.title())
print(s.capitalize())
Output:
PYTHON IS FUN
Python Is Fun
Python is fun
Explanation: These methods return new versions of the string with modified casing for various formatting needs.

s = "   hello   "
print(s.strip())
print(s.lstrip())
print(s.rstrip())
Output:
hello
hello
hello
Explanation: Strip methods remove leading and trailing whitespace, which is crucial for cleaning user input or file data.

s = "Hello, World!"
print(s.replace("World", "Python"))
Output:
Hello, Python!
Explanation: The .replace() method finds occurrences of a target phrase and swaps them with a new string.

s = "apple,banana,orange"
fruits = s.split(",")
print(fruits)
Output:
['apple', 'banana', 'orange']
Explanation: .split() breaks a single string into a list of strings based on a specific delimiter (like a comma).

fruits = ["apple", "banana", "orange"]
print(",".join(fruits))
Output:
apple,banana,orange
Explanation: The inverse of split; .join() takes a list and merges elements into one string using the provided separator.

s = "I love Python and Python loves me"
print(s.find("Python"))
print(s.count("Python"))
Output:
7
2
Explanation: .find() returns the index of the first occurrence, while .count() totals every occurrence in the string.

name = "Alice"
age = 30
print(f"Hello, {name}! You are {age} years old.")
Output:
Hello, Alice! You are 30 years old.
Explanation: F-strings (Literal String Interpolation) provide a concise way to embed expressions inside string literals using curly braces.

s = "Hello {}! Welcome to {}."
print(s.format("Bob", "Python"))
Output:
Hello Bob! Welcome to Python.
Explanation: This older formatting style uses placeholders {} that are filled by arguments passed into the .format() method.

s = "The price is $5.99"
print(s.startswith("The"))
print(s.endswith(".99"))
Output:
True
True
Explanation: Boolean methods used to verify if a string begins or ends with a specific substring, often used in conditional logic.

s = "123abc"
print(s.isalnum())
print(s.isalpha())
print(s.isdigit())
Output:
True
False
False
Explanation: These methods check if the string content is alphanumeric, purely alphabetic, or purely numeric, respectively.

s = "hello world"
print(s.swapcase())
Output:
HELLO WORLD
Explanation: Inverts the casing of every character in the string: lowercase becomes uppercase and vice versa.

s = "python programming"
print(s.center(30, "-"))
print(s.ljust(20, "*"))
print(s.rjust(20, "*"))
Output:
-----python programming------
python programming**
**python programming
Explanation: Useful for CLI table formatting, these methods pad a string with a chosen character to reach a specific width.

s = "000123"
print(s.zfill(10))
Output:
0000000123
Explanation: .zfill() adds leading zeros until the string reaches a specified length, commonly used for serial numbers or IDs.

text = "The quick brown fox jumps over the lazy dog"
print("Pangram" if all(c in text.lower() for c in "abcdefghijklmnopqrstuvwxyz") else "Not pangram")
Output:
Pangram
Explanation: This logic checks if the string contains every letter of the alphabet at least once by using a generator expression inside all().

s = "racecar"
print("Palindrome" if s == s[::-1] else "Not palindrome")
Output:
Palindrome
Explanation: A classic logic test using slicing ([::-1]) to reverse the string and compare it to the original.

s = "Hello\nWorld\t!"
print(s)
Output:
Hello
World !
Explanation: Special characters like \n (newline) and \t (tab) allow for formatting within the printed output.

s = r"Hello\nWorld\t!"
print(s)
Output:
Hello\nWorld\t!
Explanation: Prefixing a string with r tells Python to ignore escape sequences, which is essential for Windows file paths and Regular Expressions.

multiline = """Line 1
Line 2
Line 3"""
print(multiline)
Output:
Line 1
Line 2
Line 3
Explanation: Triple quotes allow a string to span multiple lines without needing the \n escape character.

s = "AbC"
print(s.lower())
print("abc".upper())
Output:
abc
ABC
Explanation: Standard methods to force a string into all lowercase or all uppercase.

email = "user@example.com"
print("Valid" if "@" in email and "." in email.split("@")[1] else "Invalid")
Output:
Valid
Explanation: A basic check using the in operator and .split() to ensure an @ symbol exists and is followed by a period.

s = "hello world hello"
print(s.rfind("hello"))
print(s.index("world"))
Output:
12
6
Explanation: .rfind() locates the last occurrence of a string, while .index() locates the first (but raises an error if not found).

s = "banana"
print(s.partition("na"))
Output:
('ba', 'na', 'na')
Explanation: Splits a string at the first occurrence of a separator and returns a 3-item tuple containing the part before, the separator, and the part after.

s = "hello"
print(" ".join(s))
Output:
h e l l o
Explanation: Demonstrates that strings are iterable; .join() can insert spaces (or any character) between every character of a string.

s = "Version 3.10"
print(s.translate(str.maketrans("0123456789", "abcdefghij")))
Output:
Version d.ab
Explanation: Uses a translation table (created by maketrans) to swap characters in bulkâ€”highly efficient for simple encryption or data cleaning.

s = "Hello World"
print(s.casefold())
Output:
hello world
Explanation: Similar to lower(), but more aggressive; it is designed to remove all case distinctions for caseless matching across different languages.

s = "123"
print(s.isnumeric())
print(s.isdecimal())
Output:
True
True
Explanation: While they seem identical for standard digits, isnumeric() is broader, returning True for things like Roman numerals or fractions in some encodings.
