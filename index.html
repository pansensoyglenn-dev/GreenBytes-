Python Encapsulation Examples

Encapsulation in Python involves bundling data (attributes) and methods that operate on the data within a class, while restricting direct access to some components (e.g., using private variables with underscores). This promotes data hiding and controlled access via getter and setter methods. Below are 10 sample programs, each with an algorithmic narrative and step-by-step explanation.

Algorithmic Narrative: Create a class to manage a bank account where the balance is private. Initialize with a balance, provide methods to deposit (add amount if positive), withdraw (subtract if sufficient funds), and get balance. Ensure direct access to balance is restricted.

Code:
class BankAccount:
    def __init__(self, initial_balance=0):
        self.__balance = initial_balance  # Private attribute

    def deposit(self, amount):
        if amount > 0:
            self.__balance += amount

    def withdraw(self, amount):
        if 0 < amount <= self.__balance:
            self.__balance -= amount

    def get_balance(self):
        return self.__balance

# Usage
account = BankAccount(100)
account.deposit(50)
account.withdraw(30)
print(account.get_balance())  # Output: 120


Step-by-Step Explanation:
1. Define the `BankAccount` class.
2. In `__init__`, set `__balance` as private (double underscore convention).
3. `deposit` checks if amount > 0, then adds to `__balance`.
4. `withdraw` checks if amount is positive and <= `__balance`, then subtracts.
5. `get_balance` returns `__balance` without allowing direct modification.
6. Create instance, perform operations, and access via getter.

Algorithmic Narrative: Model a car with private fuel level. Initialize with fuel, methods to drive (reduce fuel based on distance), refuel (add fuel), and get fuel level. Ensure fuel doesn't go negative.

Code:
class Car:
    def __init__(self, fuel=50):
        self.__fuel = fuel  # Private

        fuel_needed = distance / 10  # Assume 10 km per unit fuel

            self.__fuel -= fuel_needed

    def refuel(self, amount):
        if amount > 0:
            self.__fuel += amount

    def get_fuel(self):
        return self.__fuel

# Usage
my_car = Car()
my_car.drive(200)
my_car.refuel(30)
print(my_car.get_fuel())  # Output: 60.0


Step-by-Step Explanation:
1. Define `Car` class.
2. `__init__` sets private `__fuel`.
3. `drive` calculates fuel needed, subtracts if sufficient.
4. `refuel` adds positive amount.
5. `get_fuel` returns value.
6. Use: Drive reduces, refuel adds, get checks.

Algorithmic Narrative: Device with private battery. Init with level, charge (add up to 100), use (subtract), get level.

Code:
class Device:
    def __init__(self, battery=100):
        self.__battery = battery  # Private

    def charge(self, amount):
        if amount > 0:
            self.__battery = min(100, self.__battery + amount)

    def use(self, amount):
        if amount > 0:
            self.__battery = max(0, self.__battery - amount)

    def get_battery(self):
        return self.__battery

# Usage
dev = Device(50)
dev.use(20)
dev.charge(40)
print(dev.get_battery())  # Output: 70


Step-by-Step Explanation:
1. Class `Device`.
2. Private `__battery`.
3. `charge` adds, caps at 100.
4. `use` subtracts, floors at 0.
5. Getter.
6. Use and charge.

Python Abstraction Examples

Abstraction in Python hides complex implementation details and exposes only essential features, often using abstract base classes (ABCs) from the `abc` module. Users interact with high-level methods without knowing internals. Below are 10 samples with step-by-step explanations.

Code:
from abc import ABC, abstractmethod

class Shape(ABC):
    @abstractmethod
    def area(self):
        pass

class Circle(Shape):
    def __init__(self, radius):
        self.radius = radius

    def area(self):
        return 3.14 * self.radius ** 2

# Usage
circle = Circle(5)
print(circle.area())  # Output: 78.5


Step-by-Step Explanation:
1. Import `ABC` and `abstractmethod`.
2. Define abstract `Shape` with abstract `area`.
3. `Circle` inherits, implements `area` with formula.
4. Instantiate `Circle`, call `area`.

1. Abstract `Payment` with `process`.
2. `CreditCard` implements specific processing.
3. Create instance, call method.

Python Inheritance Examples

Inheritance allows a subclass to inherit attributes and methods from a base class, promoting code reuse. Subclasses can override or extend base behavior. Below are 10 samples with step-by-step narratives.

Python Polymorphism Examples

Polymorphism allows objects of different classes to be treated as objects of a common superclass, with methods behaving differently based on the object. Often seen with method overriding or duck typing. Below are 10 samples with step-by-step explanations.

Code:
class Shape:
    def draw(self):
        return "Drawing shape"

class Circle(Shape):
    def draw(self):
        return "Drawing circle"

class Square(Shape):
    def draw(self):
        return "Drawing square"

def render(shape):
    print(shape.draw())

# Usage
render(Circle())  # Output: Drawing circle
render(Square())  # Output: Drawing square


Step-by-Step Explanation:
1. Base `Shape` with `draw`.
2. `Circle` and `Square` override `draw`.
3. `render` function takes any Shape, calls `draw`.
4. Pass different objects, get polymorphic behavior.
